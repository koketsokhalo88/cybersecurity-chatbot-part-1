# 🔒 Cybersecurity Awareness Chatbot

A talking C# console application that teaches cybersecurity basics using Windows speech synthesis.

## Features
- 🎙️ **Text-to-Speech** - Answers spoken aloud using System.Speech
- 🎨 **ASCII Art Logo** - Visual branding with color coding
- 💬 **Interactive Chat** - Typewriter effect for responses
- 🛡️ **Cybersecurity Topics** - Password safety, phishing, malware, safe browsing

## Topics Covered
- Password safety and 2FA
- Phishing email detection
- Safe browsing habits
- Malware protection
- Social engineering awareness

## Requirements
- Windows 10/11
- .NET Framework or .NET Core
- Speech voices installed (Settings &gt; Time & Language &gt; Speech)

## How to Run
1. Clone the repository
2. Open in Visual Studio or VS Code
3. Build and run (F5)

## Code Structure
| Class | Purpose |
|-------|---------|
| `AudioPlayer` | Speech synthesis and audio management |
| `AsciiArt` | Visual logo display |
| `Chatbot` | Main conversation flow |
| `ResponseSystem` | Keyword matching and responses |
| `Utilities` | Helper methods for console output |
